Add Google Social Profiles to you Knowledge Graph Box in google search results

== Description ==

This plugin adds the required javascript for google in order to add Add Google Social Profiles to you Knowledge Graph Box in google search results .

For more information visit:

[social profiles to Google] (https://developers.google.com/webmasters/structured-data/customize/social-profiles)

== Installation ==

1. Upload `google-social-profiles` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the profiles information to the settings (Admin > Settings > Google Social Profiles)
